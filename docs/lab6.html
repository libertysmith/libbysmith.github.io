Demonstrating Shiny App: Building a Map of Texas County-Wide Polling

To demonstrate shiny app graphics, I chose to build a color-coded map of Texas County Polling Consolidation. This map should emulate the same map in this article by the Texas Tribune, “Texas conservatives want to end countywide voting. The costs could be high.” It color codes the targeted counties and maps a handful of cities for maximum interpretation power. The map is interactive so a user can check their county’s year of the switch.

County-wide polling or polling consolidation refers to the practice of reducing the number of individual polling places within a county and instead creating larger, centralized voting locations that serve multiple precincts or regions. Instead of each neighborhood or district having its own polling station, voters can cast their ballots at fewer, more centrally located places, sometimes referred to as vote centers.

This approach is intended to streamline the voting process by consolidating resources, such as staffing and equipment, and making voting locations more efficient. It can also provide more flexibility for voters, as they may be allowed to vote at any polling place within the county rather than being restricted to a specific precinct.

However, the effectiveness of county-wide polling can depend on several factors, such as transportation access, the size of the county, and voter turnout. While it may lead to better efficiency in some areas, there can also be concerns about longer wait times or reduced accessibility for voters in rural or underserved areas.

Taking data from numerous sources, the map we are building will depict which Texas counties’ currently conduct their elections with county-wide polling and which counties still run on precinct-polling. This Texas SOS website announces the counties currently switched to the vote centers process.

Building the map

# Define UI
ui <- fluidPage(
  titlePanel("Texas Counties Approved to Offer Countywide Voting"),
  leafletOutput("map", height = "800px"),  # Increase the height of the map panel
  verbatimTextOutput("county_stats")  # Output the county statistics
)

# Define Server
server <- function(input, output, session) {
  
  # Render the map with county polygons and subtle city labels
  output$map <- renderLeaflet({
    leaflet(polygons_sf) %>%
      addPolygons(layerId = ~NAME,
                  fillColor = ~ifelse(is_vote_center, "steelblue", "steelblue4"),  # Use steelblue for vote centers and steelblue4 otherwise
                  color = "lightblue", weight = 1,  # Use lightblue for the polygon borders
                  fillOpacity = 0.7, 
                  label = ~paste(NAME, "<br> Consolidation Year: ", ConsolidationYear),  # Show county name and year
                  highlightOptions = highlightOptions(weight = 3, color = "blue", bringToFront = TRUE)) %>%
      addLabelOnlyMarkers(data = city_data,  # Use addLabelOnlyMarkers to display city names
                          lat = ~Latitude, 
                          lng = ~Longitude, 
                          label = ~City,  # Always show the city names
                          labelOptions = labelOptions(noHide = TRUE, direction = 'auto', 
                                                      textsize = "12px",  # Smaller text size
                                                      opacity = 0.6,  # Reduced opacity for subtlety
                                                      style = list("color" = "gray"))) %>%
      setView(lng = -99.341, lat = 31.9686, zoom = 6)  # Center the view on Texas
  })
  
  # React to clicks on the map
  observeEvent(input$map_shape_click, {
    clicked_county <- input$map_shape_click$id  # Get the clicked county's NAME
    
    # Filter the data for the clicked county and display statistics
    county_data <- polygons_sf %>% filter(NAME == clicked_county)
    
    # Output the statistics (for example, SHAPE_Area and SHAPE_Leng)
    output$county_stats <- renderText({
      paste("County:", clicked_county, 
            "\nArea:", round(county_data$SHAPE_Area, 2), 
            "\nPerimeter Length:", round(county_data$SHAPE_Leng, 2))
    })
  })
}

# Run the Shiny app
shinyApp(ui = ui, server = server)
